A classic Snake game built with HTML, CSS, and JavaScript. Control the snake, eat food, and try to achieve the highest score without hitting the walls or yourself!

## Features

- **Classic Gameplay**: Navigate the snake using arrow keys
- **Score Tracking**: Keep track of your current score and high score
- **Responsive Controls**: Start and reset the game with buttons

## Controls

- **Start Game**: Begin a new game
- **Reset**: Reset the game to start over

## Game Rules

- Each piece of food eaten adds 10 points to your score
- The snake grows longer each time it eats food
- The game ends if the snake hits:
  - The wall (canvas borders)
  - Its own body
- High scores are automatically saved in your browser
